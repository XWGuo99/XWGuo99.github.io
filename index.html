Hello, I’m Xinwei.

Graduate researcher at MSU, focusing on LiDAR–Camera sensor fusion, self-calibration, and 3D perception. Previously worked in the LiDAR industry on deployment, calibration, and performance evaluation. Currently developing DA-LCCNet, a domain-adaptive calibration network in PyTorch.

About

I aim to bridge real‑world multi‑modal sensing and deep learning to build deployable autonomous driving perception systems. My recent focus includes targetless LiDAR‑Camera self‑calibration, domain adaptation, and analyzing how calibration error impacts 3D object detection.

Keywords: Sensor Calibration, Domain Adaptation, Object Detection, Data Processing

Research Topics

  • LiDAR–Camera Self-Calibration: Conducted a comprehensive review of existing calibration methods and evaluated their accuracy by testing calibration errors through a 3D object detector.
  • Sensor Fusion Domain Adaptation: Developed DA-LCCNet, an enhanced model with domain-adaptation modules that significantly reduce calibration errors under domain-shift conditions, improving 3D object detection performance by over 50%.
